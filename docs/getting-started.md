# Getting Started with AdvisorLeads

This tutorial walks you through everything you need to go from a fresh install to finding your first set of advisor candidates. By the end you will have:

- downloaded and launched AdvisorLeads on your Windows PC
- waited for the initial data load to complete
- navigated the main interface
- filtered advisors by state, type, and experience
- reviewed a full advisor profile
- saved a favorite and built a named list

You do not need any technical background. The only things you need are a Windows 10 or later PC and an internet connection.

---

## 1. Download and Launch

### Download the app

1. Open your web browser and go to the [AdvisorLeads Releases page](https://github.com/dannymayer/AdvisorLeads/releases).
2. Under the latest release, download **AdvisorLeads-win-x64.zip**.
3. Open your Downloads folder, right-click the zip file, and select **Extract All…**
4. Choose a folder you will remember — for example `C:\AdvisorLeads` — and click **Extract**.

### Run the app

Open the extracted folder and double-click **AdvisorLeads.exe**. Windows may show a security prompt the first time ("Windows protected your PC"). Click **More info**, then **Run anyway** — this is expected for applications that are not installed through the Microsoft Store.

---

## 2. The First Launch — Let It Download

The very first time you run AdvisorLeads, it automatically downloads regulatory data from FINRA and the SEC. You will see a progress bar and status messages at the bottom of the window.

**This is normal.** The download can take anywhere from two to ten minutes depending on your internet connection. The app is pulling in the full national database of registered financial advisors and investment adviser firms — typically several hundred thousand records.

> **Tip:** Leave the window open and let it finish. You can check back after a few minutes. Do not close the app during this step or you will need to wait again next time.

Once the status bar reads something like **"Ready — X advisors loaded"**, the download is complete and you are ready to search.

## 3. A Tour of the Interface

The main window has three main areas:

```
┌─────────────────────────────────────────────────────────────────────┐
│  [Dashboard] [Individuals] [Firms] [Reports] [Alerts]   ← tabs      │
├──────────────┬────────────────────────────┬────────────────────────┤
│              │                            │                         │
│  Filter      │    Results list            │   Detail panel          │
│  panel       │    (center)                │   (right side)          │
│  (left)      │                            │                         │
└──────────────┴────────────────────────────┴────────────────────────┘
```

**Tabs across the top**

| Tab | What it shows |
|-----|---------------|
| Dashboard | A summary snapshot — total advisors and firms in the database, recent activity |
| Individuals | Search and browse individual financial advisors |
| Firms | Search and browse investment adviser firms |
| Reports | Pre-built analytical reports useful for recruiting |
| Alerts | Notifications when watched advisors or firms change |

**Filter panel (left side)** — on the Individuals and Firms tabs, this panel lets you narrow down the results using criteria like state, name, firm, and experience level.

**Results list (center)** — shows the advisors or firms that match your current filters. Click any row to select it.

**Detail panel (right side)** — shows the full profile for whichever advisor or firm is selected in the center list.

## 4. Finding Your First Advisors

Click the **Individuals** tab to get started.

### Step 1: Filter by state

In the filter panel on the left, find the **State** dropdown. Click it and select a state — for example, **TX** for Texas. The results list in the center will update automatically to show only advisors registered in that state.

> **Tip:** The results count is shown in the filter panel header badge. If you see "0", try a different state or clear other filters first.

### Step 2: Filter by record type

The **Record Type** dropdown lets you focus on the kind of advisor you are recruiting:

| Record Type | Plain English |
|-------------|---------------|
| Investment Advisor Representative | An advisor registered with the SEC or a state as an investment adviser — typically fee-based, manages client portfolios |
| Registered Representative | A broker registered with FINRA — typically commission-based, sells securities products |
| (All Types) | Shows both |

Select the type that matches your recruiting focus.

### Step 3: Filter by experience

The **Min Exp (yr)** and **Max Exp (yr)** fields let you set a range. For example, entering `10` in the **Min Exp** field will return only advisors with at least ten years in the industry. Leave **Max Exp** blank to set no upper limit.

### Step 4: Filter by firm name (optional)

If you are looking for advisors who currently work at — or have worked at — a specific firm, type part of the firm name in the **Firm** box. The filter matches partial names, so typing `Merrill` will find advisors at Merrill Lynch.

### Step 5: Read the results

The center panel shows a list of matching advisors. Each row shows:

- **Name** — last name, first name
- **Current Firm** — where they work now
- **State** — their registered state
- **Years Exp** — years in the industry
- **Disclosures** — whether there are any regulatory disclosures on file

Rows highlighted in amber indicate advisors you have previously marked as favorites. Rows with a disclosure flag are shown in a different color to make them easy to spot.

## 5. Reviewing an Advisor's Full Profile

Click on any advisor row in the results list. Their full profile appears in the detail panel on the right side of the screen.

### The summary section

At the top of the detail panel you will see:

- **Full name**, CRD number (a unique industry identifier), and current registration status
- **Current firm** and how long they have been there
- **Total number of firms** they have worked for in their career
- **Any active disclosure or sanction badges** — shown as colored labels if present

### The detail tabs

Below the summary, four tabs give you deeper information:

**Employment History** — A list of every firm this advisor has been registered with, in reverse chronological order. This tells you their career path and how frequently they have changed firms.

**Disclosures** — A list of any regulatory actions, customer complaints, arbitration awards, or criminal disclosures on their FINRA or SEC record. Most advisors have zero disclosures; if this tab shows any, review them carefully before reaching out.

**Qualifications** — Licenses and professional certifications on file — for example, Series 7, Series 65, CFP, CFA.

**Registrations** — The regulatory authorities (states and/or FINRA) where this advisor is currently registered, and the scope (broker-dealer vs. investment adviser).

> **What are "disclosures"?** In the financial industry, a disclosure is a formal regulatory event that has been reported to FINRA or the SEC. Common examples include customer complaints, civil lawsuits, regulatory sanctions, or criminal charges. Disclosures do not automatically disqualify a candidate — but they are an important part of due diligence before recruiting.

## 6. Saving Favorites and Building a List

### Mark an advisor as a favorite

When you find an advisor you want to keep track of, click the **☆ Favorite** button in the detail panel header. The star fills in to show the advisor is now saved. You can later filter to **Favorites Only** in the filter panel to see all your starred advisors in one place.

### Create a named list

Named lists let you organize advisors into groups — for example, "Q2 Outreach", "Potential RIA Converts", or "Follow Up".

1. Click the **Lists** tab in the main menu bar at the top.
2. Click **New List** and give it a name.
3. Go back to the **Individuals** tab, find an advisor you want to add, and right-click their row in the results list.
4. Select **Add to List…** and choose the list you just created.

You can also add an advisor to a list directly from their detail panel using the **Add to List** button.

To view or manage your lists, return to the **Lists** tab at any time.

## 7. What to Do Next

You have now covered the core workflow. Here are the next things to explore:

**Fetch fresh data for a specific advisor or firm**
The background refresh runs automatically every hour, but you can pull the latest data right now. Go to **Data → Fetch New Data…** in the menu bar and enter a name, CRD number, or firm. This queries FINRA and the SEC in real time.

**Browse firms**
Click the **Firms** tab to search for investment adviser firms directly. You can filter by state, AUM range, number of advisors, and compensation model (fee-only, commission-based, etc.). Clicking a firm in the results list shows its full Form ADV profile on the right.

**Run a recruiting report**
The **Reports** tab contains pre-built analyses designed for BD and RIA recruiters — including lists of high-mobility advisors, firms with shrinking headcounts, and advisors who match specific production profiles. Select a report from the list on the left and click **Run Report**.

**Export to Wealthbox CRM**
If your firm uses Wealthbox CRM, you can push advisor contacts directly from AdvisorLeads. Right-click an advisor in the results list and select **Export to Wealthbox CRM**. You will need to enter your Wealthbox API token first — go to **Tools → Wealthbox CRM Settings** to set that up.

**Set up alerts**
The **Alerts** tab lets you watch specific advisors or firms and receive notifications when something changes — for example, when a watched advisor moves to a new firm, or when a firm's AUM crosses a threshold you have defined.

## Quick Reference: Filter Panel Controls

| Control | What it does |
|---------|--------------|
| Name | Searches first name, last name, or full name (partial match) |
| CRD # | Look up an advisor by their unique CRD number |
| State | Filter to advisors registered in a specific state |
| City | Filter to a specific city |
| Firm | Filter by current or previous firm name (partial match) |
| Record Type | Investment Advisor Representative, Registered Representative, or both |
| Min / Max Exp (yr) | Years of industry experience range |
| Status | Active, Inactive, or both |
| Source | FINRA-sourced, SEC-sourced, or both |
| Disclosures | Show all, only those with disclosures, or only those without |
| Favorites Only | Show only advisors you have starred |
